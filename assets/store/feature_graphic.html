📸 フィーチャーグラフィック作成手順

  1. ブラウザで全画面表示: F11キー（Mac: Control + Command + F）
  2. ズームを100%に設定: Cmd + 0（Mac）/ Ctrl + 0（Windows）
  3. スクリーンショット撮影:
    • Mac: Cmd + Shift + 4 → スペースキー → グラフィック部分をクリック
    • Windows: Snipping Tool または Win + Shift + S
  4. サイズ確認: 1024 x 500 ピクセルであることを確認
  5. 保存: PNG形式で feature_graphic.png として保存
